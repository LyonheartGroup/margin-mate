# MarginMate.ai

MarginMate.ai is a static pricing and profitability calculator for freelancers, consultants, and boutique agencies.

## What it does

- Calculates delivery cost
- Adds revision and overhead buffers
- Shows expected profit and profit margin
- Calculates a minimum viable quote
- Recommends a quote based on target margin
- Flags the deal as healthy, tight, or bad
- Copies a summary for proposals or CRM notes

## Files

- `index.html` — landing page and calculator UI
- `styles.css` — visual design and responsive layout
- `script.js` — calculator logic
- `pack/` — starter files for the paid Pricing Pack

## Deploy

This is a plain static site. Deploy it to GitHub Pages, Cloudflare Pages, Netlify, or any static host.

### GitHub Pages quick start

1. Push the repo to GitHub.
2. In GitHub, open **Settings → Pages**.
3. Set source to **Deploy from a branch**.
4. Choose branch **main** and folder **/(root)**.
5. Save.

## Payment link
