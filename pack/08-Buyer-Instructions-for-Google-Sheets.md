# Buyer Instructions — Using the workbook files in Google Sheets

The workbook files in this pack are provided as CSV templates so they are easy to use in Google Sheets.

## How to use them
1. Open Google Sheets
2. Create a new spreadsheet
3. Go to **File → Import**
4. Upload the CSV file you want to use
5. Choose whether to create a new sheet or replace the current one
6. Start filling in your own numbers and notes

## What to use each file for
- **Advanced Pricing Sheet** — compare project pricing scenarios
- **Proposal Pricing Workbook** — pressure-test a quote before sending it
- **Retainer Pricing Workbook** — price recurring work more cleanly
- **Service Suite Planner** — separate offers and scope clearly
- **Client Pricing Scenario Workbook** — compare multiple client pricing options
- **Discount Protection Workbook** — handle discount requests without wrecking margin
- **Capacity and Profit Workbook** — estimate growth capacity and profit by month
- **Profitability Review Workbook** — review what a project actually earned after delivery

## Important note
These files are templates. Use them as working tools, not rigid rules. Pricing still requires judgment, context, and clear scope.
