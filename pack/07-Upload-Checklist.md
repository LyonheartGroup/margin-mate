# Upload Checklist

Before publishing the Gumroad product:

- Confirm the product name is **MarginMate Pricing Pack**
- Set the price to **$19**
- Upload the final zip file
- Paste the short description and full description from `06-Gumroad-Product-Copy.md`
- Add a cover image if available
- Confirm the checkout/delivery flow works
- Copy the product URL
- Replace the placeholder button on the site with the real Gumroad URL

After publishing:
- Test the checkout link in a private browser window
- Verify the buyer receives the file automatically
- Verify the thank-you page works
- Turn on GitHub Pages / custom domain if not already done
